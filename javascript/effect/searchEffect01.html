CSS 속성 검색하기

CSS 속성 갯수 : 0
  • accent-color : href특성을 통해 다른 페이지나 같은페이지의 위치, 파일와 그 외 다른 URL로 연결할 수 있는 하이퍼 링크를 만듭니다.
  • align-content : 콘텐츠 아이템의 상하관계 정렬 상태를 설정
  • align-items : 콘텐츠 아이템의 내부 상하관계 정렬 상태를 설정
  • align-self : 개별적인 콘텐츠 아이템의 정렬 상태를 설정
  • all : 요소의 속성을 초기화 또는 상속을 설정
  • animation : 애니메이션과 관련된 속성을 일괄적으로 설정
  • animation-delay : 애니메이션 지연 시간을 설정
  • animation-direction : 애니메이션 움직임 방향을 설정
  • animation-duration : 애니메이션 움직임 시간을 설정
  • animation-fill-mode : 애니메이션이 끝난 후의 상태를 설정
  • animation-iteration-count : 애니메이션의 반복 횟수를 설정
  • animation-name : 애니메이션 Keyframe 이름을 설정
  • animation-play-state : 애니메이션 진행 상태를 설정
  • animation-timeline : 애니메이션을 설명하는 하나 이상의 규칙과 이름을 설정
  • animation-timing-function : 애니메이션 움직임의 속도를 설정
  • appearance : 운영 체제의 테마를 기반으로 하는 UI 컨트롤의 기본 모양을 제어
  • aspect-ratio : 자동 크기 및 기타 레이아웃 기능 계산에 사용되는 상자의 기본 종횡비를 설정
  • backdrop-filter : 배경에 그래픽 효과를 설정
  • backface-visibility : 요소의 뒷면을 설정합니다. 요소가 3D인 경우에서 설정 할 수 있다.
  • background : 백그라운드 속성을 일괄적으로 설정
  • background-attachment : 배경 이미지의 고정 여부를 설정
  • background-blend-mode : 배경을 혼합했을 때 그래픽 효과를 설정
  • background-clip : 백그라운드 이미지의 위치 기준점을 설정
  • background-color : 백그라운드 색을 설정
  • background-image : 백그라운드 이미지 속성을 설정
  • background-origin : 백그라운드 이미지의 위치 기준점을 설정하기 위한 속성
  • background-position : 백그라운드 이미지의 위치 영역을 설정
  • background-position-x : 백그라운드 이미지의 x축 위치 영역을 설정
  • background-position-y : 백그라운드 이미지의 y축 위치 영역을 설정
  • background-repeat : 백그라운드 이미지 반복 여부를 설정
  • background-size : 백그라운드 이미지 사이즈를 설정
  • block-size : 쓰기 모드에 따라 요소 블록의 가로 또는 세로 크기를 설정
  • border : 테두리 속성을 일괄적으로 설정
  • border-block : 테두리 속성을 고정
  • border-block-color : 고정된 테두리 속성에 색상 부여
  • border-block-end : 테두리 속성 끝 부분에 고정
  • border-block-end-color : 테두리 속성 끝 부분에 고정 후 색상 부여
  • border-block-end-style : 테두리 속성 끝 부분에 고정 후 스타일 부여
  • border-block-end-width : 테두리 속성 끝 부분에 고정 후 width 속성 부여
  • border-block-start : 테두리 속성 시작 부분에 고정

    // 선택자
    const quizType = document.querySelector(".quiz__type");                     // 퀴즈 종류
    const quizNumber = document.querySelector(".quiz__question .number");       // 퀴즈 번호
    const quizAsk = document.querySelector(".quiz__question .ask");             // 퀴즈 질문
    const quizConfirm = document.querySelector(".quiz__answer .confirm");     // 정답 확인 버튼
    const quizResult = document.querySelector(".quiz__answer .result");       // 정답 결과
    const quizDog = document.querySelector(".quiz__view .dog");                 // 강아지
    
    // 문제 정보
    const answerType = "웹디자인기능사";
    const answerNum = 1;
    const answerAsk = "인접하는 두 색의 경계 부분에 색상, 명도, 채도의 대비가 더욱 강하게 일어나는 현상을 무엇이라고 하는가?";
    const answerResult = "연변대비";

    // 문제 출력
    quizType.innerText = answerType;
    quizNumber.innerText = answerNum + ".";
    quizAsk.innerText = answerAsk;
    quizResult.innerText = answerResult;

    // 정답 숨기기
    quizResult.style.display = "none";

    // 정답 확인
    // 정답 버튼 클릭 시 확인 버튼을 안보이게.. 숨겨지 정답은 보이게..
    // 정답을 확인하면 강아지가 웃고 있어여 함  // 클래스 like를 추가
    quizConfirm.addEventListener("click", function(){
        quizConfirm.style.display = "none";
        quizResult.style.display = "block";
        quizDog.classList.add("like");
    });
