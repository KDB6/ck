01. 배열 메서드 : join() : 배열 요소 문자열 결합 : 반환(문자열)


     const arrNum = [100, 200, 300, 400, 500];
    const text1 = arrNum.join('');
    const text2 = arrNum.join(' ');
    const text3 = arrNum.join('*');
    const text4 = arrNum.join('-');
text1 100200300400500
text2 100 200 300 400 500
text3 100*200*300*400*500
text4 100-200-300-400-500

02. 배열 메서드 : push()

push() 메서드는 배열 마지막 요소에 추가 : 반환(숫자)


    const arrNum = [100, 200, 300, 400, 500]
    const arrPush = arrNum.push(600);

    document.querySelector(".sample01_M1").innerHTML = "push(600)";
    document.querySelector(".sample01_P1").innerHTML = arrPush; // 6
    document.querySelector(".sample01_A1").innerHTML = arrNum;  // 100, 200 , 300, 400, 500, 600
메서드 리턴값 결과값
push600 6 100, 200, 300, 400, 500, 600

03. 배열 메서드 : pop()

pop() 메서드는 배열 마지막 요소를 삭저 : 변환(삭제된 요소)


    const arrNum2 = [100, 200, 300, 400, 500]
    const arrPush2 = arrNum2.pop();

    document.querySelector(".sample03_M1").innerHTML = "pop()";
    document.querySelector(".sample03_P1").innerHTML = arrPush2; // 500
    document.querySelector(".sample03_A1").innerHTML = arrNum2;  // 100, 200 , 300, 400
메서드 리턴값 결과값
pop() 500 100, 200, 300, 400
